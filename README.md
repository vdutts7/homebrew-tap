# homebrew-tap

Homebrew tap for vd7 tools.

## Install

```bash
brew tap vdutts/tap
brew install applock
```

## Formulas

| Formula | Description |
|---------|-------------|
| `applock` | Protect any macOS app behind Touch ID |
