# homebrew-tap

Homebrew tap for vd7 tools.

## Install

```bash
brew tap vdutts7/tap
brew install applock
```

## Formulas

| Formula | Description | Repo |
|---------|-------------|------|
| `applock` | Protect any macOS app behind Touch ID | [applock-macos](https://github.com/vdutts7/applock-macos) |
